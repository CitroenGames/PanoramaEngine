# PanoramaEngine Documentation

This folder holds the build, integration, architecture, and feature reference
for PanoramaEngine.

## Guides

- [Building](building.md) - prerequisites, `sighmake` commands, targets, and
  validation notes.
- [Integration](integration.md) - resource providers, document loading, runtime
  setup, input, animation, layout, and rendering.
- [Architecture](architecture.md) - pipeline overview, module map, ownership,
  lifetime, and extension points.
- [Panorama Support](panorama-support.md) - supported Panorama document features,
  CSS, scripting, rendering, input behavior, and current limits.

## Best Starting Points

Read these in order when integrating the library into an application:

1. [Building](building.md)
2. [Integration](integration.md)
3. [Panorama Support](panorama-support.md)

Read [Architecture](architecture.md) when changing library internals or adding
extension points.
